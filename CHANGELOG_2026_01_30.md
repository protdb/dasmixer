# Changelog - 2026-01-30

## Новый функционал

### ✨ Удаление групп и инструментов

- **Удаление групп (Comparison Groups)**: Добавлена кнопка удаления для каждой группы с проверкой наличия связанных образцов
- **Удаление инструментов (Tools)**: Добавлена кнопка удаления для каждого инструмента с проверкой наличия идентификаций
- **Диалоги подтверждения**: Перед удалением показывается диалог с предупреждением о необратимости операции
- **Обработка ошибок**: Если есть зависимые данные, показывается информативное сообщение

### ✨ Изменение группы образца

- **Dropdown в списке образцов**: Каждый образец теперь имеет выпадающий список для выбора группы
- **Быстрое переназначение**: Группу можно изменить одним кликом без дополнительных диалогов
- **Автоматическое обновление**: После изменения счетчики образцов в группах обновляются автоматически
- **Опция "None"**: Можно убрать образец из всех групп

## Использование

### Удалить группу
1. Нажать иконку корзины справа от группы
2. Подтвердить в диалоге
3. Группу можно удалить только если в ней нет образцов

### Удалить инструмент
1. Нажать иконку корзины справа от инструмента
2. Подтвердить в диалоге
3. Инструмент можно удалить только если нет связанных идентификаций

### Изменить группу образца
1. В списке образцов найти нужный
2. Выбрать новую группу из выпадающего списка справа
3. Группа изменится автоматически
