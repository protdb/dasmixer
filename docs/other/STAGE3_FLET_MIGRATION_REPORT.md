## Проблема

В процессе работы с этапом 3 было выявлено, что часть UI не работает корректно. В частности:

- ❌ Не работал диалог выбора MGF-файлов
- ❌ Не работал диалог выбора папки для pattern matching
- ❌ Некорректно отображались уведомления (SnackBar)
- ❌ Диалоги не открывались/закрывались должным образом

**Причина:** Код был написан с учётом устаревшей версии Flet. В текущей версии (0.80.4) API существенно изменился.

## Выполненная работа

### 1. Анализ документации
✅ Изучена актуальная документация Flet через Context7  
✅ Выявлены все изменения API между версиями  
✅ Составлен план миграции

### 2. Исправление FilePicker API

**Было (старый API):**
```python
file_picker = ft.FilePicker(on_result=callback)
page.overlay.append(file_picker)
file_picker.pick_files(...)
```

**Стало (новый API):**
```python
files = await ft.FilePicker().pick_files(
    dialog_title="Select Files",
    allowed_extensions=["mgf"],
    allow_multiple=True
)
```

**Применено в:**
- `show_import_single_files()` - выбор MGF файлов
- `show_import_pattern_dialog()` - выбор папки через `get_directory_path()`

### 3. Исправление AlertDialog API

**Было:**
```python
page.show_dialog(dialog)
page.pop_dialog()
```

**Стало:**
```python
page.overlay.append(dialog)
dialog.open = True
page.update()

# Закрытие:
dialog.open = False
page.update()
```

**Применено во всех диалогах:**
- Add Group Dialog
- Import Mode Selection Dialog
- Pattern Matching Dialog
- Single Files Config Dialog
- Progress Dialog

### 4. Исправление SnackBar API

**Было:**
```python
page.show_dialog(ft.SnackBar(...))
```

**Стало:**
```python
page.snack_bar = ft.SnackBar(
    content=ft.Text("Message"),
    bgcolor=ft.Colors.GREEN_400
)
page.snack_bar.open = True
page.update()
```

**Применено во всех уведомлениях.**

### 5. Исправление Button атрибутов

**Было:**
```python
ft.ElevatedButton(text="Click me", ...)  # ❌ Неверный атрибут
```

**Стало:**
```python
ft.ElevatedButton(content="Click me", ...)  # ✅ Правильно
```

**Применено во всех кнопках в файле `samples_tab.py`.**
