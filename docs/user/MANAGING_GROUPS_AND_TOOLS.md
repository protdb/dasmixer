## Управление группами сравнения

Группы сравнения (Comparison Groups) используются для организации образцов и проведения сравнительного анализа.

### Создание группы

1. Откройте вкладку **Samples**
2. В секции **Comparison Groups** нажмите кнопку **Add Group**
3. Заполните поля:
   - **Group Name** - уникальное название группы (обязательно)
   - **Description** - описание группы (необязательно)
   - **Color** - цвет для визуализации в hex формате (например, `3B82F6`)
4. Нажмите **Add**

### Просмотр групп

В секции **Comparison Groups** отображаются все созданные группы с информацией:
- Название группы
- Количество образцов в группе
- Описание группы (если указано)
- Цветная иконка папки

### Удаление группы

1. В списке групп найдите группу, которую хотите удалить
2. Нажмите на **иконку корзины** (🗑️) справа от группы
3. В появившемся диалоге прочитайте предупреждение
4. Нажмите **Delete** для подтверждения или **Cancel** для отмены

**⚠️ Важно:**
- Группу можно удалить только если в ней **нет образцов**
- Если в группе есть образцы, появится сообщение: *"Cannot delete: X samples are associated with it"*
- Сначала переместите все образцы в другие группы или уберите группу у образцов
- Операция удаления **необратима**

## Управление инструментами идентификации

Инструменты (Tools) представляют методы идентификации пептидов (de novo секвенирование, поиск по базе данных и т.д.).

### Создание инструмента

1. Откройте вкладку **Samples**
2. В секции **Identification Tools** нажмите кнопку **Add Tool**
3. Заполните поля:
   - **Tool Name** - название инструмента (например, "PowerNovo2", "MaxQuant")
   - **Parser / Format** - выберите формат файлов идентификации из списка
   - **Color** - цвет для визуализации (например, `9333EA`)
4. Нажмите **Add**

### Просмотр инструментов

В секции **Identification Tools** отображаются все инструменты с информацией:
- Название инструмента
- Количество импортированных файлов идентификации
- Тип парсера (формат файлов)
- Кнопка для импорта идентификаций

### Удаление инструмента

1. В списке инструментов найдите инструмент, который хотите удалить
2. Нажмите на **иконку корзины** (🗑️) справа от кнопки "Import Identifications"
3. В появившемся диалоге прочитайте предупреждение
4. Нажмите **Delete** для подтверждения или **Cancel** для отмены

**⚠️ Важно:**
- Инструмент можно удалить только если с ним **не связаны идентификации**
- Если есть идентификации, появится сообщение: *"Cannot delete: X identifications are associated with it"*
- Если вы удалите инструмент, все его настройки будут потеряны
- Операция удаления **необратима**

## Изменение группы образца

### Как изменить группу

1. Откройте вкладку **Samples**
2. В секции **Samples** найдите нужный образец
3. Справа в строке образца находится выпадающий список **Group**
4. Выберите новую группу из списка
5. Группа образца изменится **автоматически**

### Опции выбора группы

- **Группы** - все созданные группы сравнения
- **None** - убрать образец из всех групп (образец будет без группы)

### Что происходит при изменении

- Группа образца обновляется в базе данных
- Счетчики образцов в группах автоматически пересчитываются
- Появляется уведомление: *"Updated sample 'Sample_001' group to 'Control'"*
- Изменение применяется **немедленно**, отмены нет

### Массовое изменение групп

Для изменения группы у нескольких образцов:
1. Измените группу у первого образца
2. Повторите для остальных образцов

*Примечание: Функция массового изменения будет добавлена в будущих версиях*

## Часто задаваемые вопросы

### Почему я не могу удалить группу?

**Ответ:** Группу можно удалить только если в ней нет образцов. Сначала:
1. Посмотрите, сколько образцов в группе (показано в описании группы)
2. Измените группу у всех образцов на другую или уберите группу (выберите "None")
3. После этого группу можно будет удалить

### Почему я не могу удалить инструмент?

**Ответ:** Инструмент можно удалить только если с ним не связаны идентификации. Это сделано для предотвращения потери данных. Если вам действительно нужно удалить инструмент с идентификациями, обратитесь к разработчику или используйте API для ручного удаления.

### Можно ли восстановить удаленную группу или инструмент?

**Ответ:** Нет, операция удаления необратима. Перед удалением всегда показывается диалог подтверждения. Будьте внимательны!

### Что произойдет с образцом, если я уберу его группу?

**Ответ:** Образец останется в проекте, но не будет участвовать в сравнительном анализе между группами. Его можно будет использовать для других целей или назначить группу позже.

### Можно ли переименовать группу или инструмент?

**Ответ:** В текущей версии функция редактирования названий недоступна. Эта функция будет добавлена в следующих обновлениях.

### Как узнать, сколько образцов в группе?

**Ответ:** Количество образцов отображается в описании группы, например: *"5 samples • Control group"*

### Могу ли я создать группу с таким же названием после удаления?

**Ответ:** Да, после удаления группы ее название освобождается и может быть использовано снова.

### Что делать, если случайно изменил группу образца?

**Ответ:** Просто выберите правильную группу из выпадающего списка еще раз. Изменение применяется сразу, но вы можете изменить его обратно в любой момент.

### Сколько групп можно создать?

**Ответ:** Технически ограничений нет, но для удобства работы рекомендуется использовать разумное количество групп (обычно 2-10).

### Можно ли экспортировать список групп и образцов?

**Ответ:** Информация о группах и образцах сохраняется в файле проекта (.dasmix). Функция экспорта в отдельный файл будет добавлена в будущих версиях.

## Советы и рекомендации

### Организация групп

1. **Используйте понятные названия**: "Control", "Treatment_1h", "Treatment_24h" вместо "Group1", "Group2"
2. **Добавляйте описания**: Укажите важные детали в поле Description
3. **Используйте цвета**: Разные цвета помогут быстро визуально различать группы
4. **Планируйте заранее**: Создайте все необходимые группы перед импортом образцов

### Работа с инструментами

1. **Четкие названия**: Укажите версию и параметры, например "PowerNovo2_v1.0" или "MaxQuant_Oxidation"
2. **Один инструмент - один формат**: Не используйте один инструмент для разных форматов файлов
3. **Тестируйте на одном файле**: Перед массовым импортом проверьте настройки на одном файле

### Управление образцами

1. **Назначайте группы сразу**: При импорте спектров указывайте группу
2. **Проверяйте перед анализом**: Убедитесь, что все образцы правильно распределены по группам
3. **Документируйте**: Используйте систему именования образцов, которая отражает их группу
