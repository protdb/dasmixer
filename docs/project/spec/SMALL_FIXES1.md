## Окно настроек

Настройки Batch из меню "настройки" не применяются к реальным размерам батчей. Нужно поправить, связать параметры в настройках к функциям.
Нюанс: весь проброс параметров должен идти через GUI, обращение к конфигу внутри модулей dasmixer/api/calculations нежелательно

Добавить настройку Max CPU threads, выполняемых с помощью Multiprocessing. По умолчанию определяется как сейчас (os.cpu_count - 1)

Добавить настройки логирования: флаг "Log operations to file", log level (dropdown: DEBUG, INFO, WARNING, ERROR), path to logs folder.  Привязать настройки к системному логгеру.

### Перемещение кнопок

В секции Actions убрать кнопки из выпадающего блока, сделать их положение фиксированным.

Оставить, получается, два вертикальных блока в этом разделе:
1) Ion Matching settings
2) Actions + Protein sequence Library

В Actions переименовать кнопку Run Identification Matching в Select Preferred (с иконкой ft.Icons.STAR_OUTLINE)

Добавить в Actions кнопку Save settings: по кнопке должны сохраняться указанные пользователем параметры идентификации в момент нажатия кнопки (логика сохранения при запуске остается)

Добавить "сворачивание" блока Tool Settings при переходе на другую вкладку. Вопрос производительности, насыщенный контролами блок. При сворачивании: сохраняем параметры (фоном), удаляем секцию из Flet, см. логику для Table And Plot View.
