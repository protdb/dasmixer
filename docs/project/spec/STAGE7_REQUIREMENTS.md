## Доработки UI

Функционал определения preferred identifications

На UI (вкладка peptides) и в логике должны отражаться следующие параметры для инструмента (Tool settings):
- Min top-10 peaks covered (по умолчанию 1; ключ: min_top_peaks)
- Min ions covered in spectra (по умолчанию 5; ключ: min_ions_covered)
- Min spectre peaks (по умолчанию 10; ключ: min_spectre_peaks)
- Use Leucine Combinatorics (чекбокс, по умолчанию False; ключ: leucine_combinatorics)
 
Параметры должны обрабатываться аналогично остальным параметрам, передаваться в соответствующие
функции при вызове вместе с остальными параметрами. Сейчас функции matching'а белков и идентификаций 
уже используют эти параметры вместе с остальными параметрами инструментов.
