# Дораотки после релиза (статьи)

## Мелкие правки
- Возможность перезагрузить файл идентификаций, изменить путь и втянуть идентификации - пониженный приоритет
- Seqence selection criteria: вернуть, сделать корректные enum'ы в коде - пониженный приоритет

## Доработки CLI
- Режим CLI для Windows как отдельная сборка
- Расширенный функционал: формирование отчетов, выполнение расчетов, экспорт данных

## Импорт/экспорт

Импорт:
- mzML
- PEAKS
- mzTab

Экспорт:
- Выгрузка для ProteomeXchange с загрузкой на FTP

## Оптимизация
- Вкладка Samples, расчёт параметров для всех образцов сразу по каждому параметру
- Вынос массовых расчетов в компилируемый код (C/Numba?). Частичная замена Pyteomics и Peptacular на собственное решение с повышенной производительностью

## Архитектура
- Разделение на три условно-независимых пакета:
  - dasmixer-api: расчеты, работа с проектом dasmix, интеграция через Python
  - dasmixer-cli: интерфейс командной строки для процессинга данных на сервере
  - dasmixer-gui: полная версия приложения для десктопа
- Выделение версии DASMixer-server для переноса вычислений на отдельную машину, использование GUI как тонкого клиента, сборка GUI для чистого Web(?)
